General Information

GeoGuru is a travel planner where users can create their own itinerary. Users enter their desired location of travel, conditions of times to go out, and theme of activities, and this application will provide a list of nearby events that the user can then plan and add to their itinerary. The application requires that users create an account, which allows it to save each user's vacation plans across different devices.
